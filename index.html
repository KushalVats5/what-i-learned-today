with spread operator 
	
		try {
			let impPart = ['keyboard', '2gb data'];
			let whatINeed = ['good sleep', 'laptop', ...impPart, 'repeat'];

			console.log(whatINeed);
		} catch(ourFirend) {
			console.log(ourFirend);
		}

3) Get event attached to current node in chrome step
  • Open console
  • select element tab (left most side)
  • Click on Event Listeners tab (right most side)
  • Un check Ancestors box to get only the events attached to current node we selected
